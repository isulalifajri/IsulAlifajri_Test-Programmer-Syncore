1. Menampilkan data member yang berada pada provinsi sumatera utara dan dalam satu kabupaten

Jawab

- Menggunakan Metode Elequent laravel
$members = Member::where('id_provinsi', 2) ->orWhereHas('instansi.kecamatan.kabupaten', function ($query) { $query->where('id', 11.05); }) ->orderBy('created_at', 'DESC') ->paginate(5);

2. Menampilkan data provinsi yang tidak ada dalam data member

Jawab

- Menggunakan Metode Elequent laravel
$provinsis = Provinsi::whereNotIn('id', function ($query) { $query->select('id_provinsi') ->from('members') ->distinct(); }) ->get();

3. Menampilkan data kabupaten yang tidak ada dalam data member

Jawab

- Menggunakan Metode Elequent laravel
$kabupatens = Kabupaten::whereNotIn('id', function ($query) { $query->select('id_kabupaten') ->from('members') ->distinct(); }) ->get();
